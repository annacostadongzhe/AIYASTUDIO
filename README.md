 # AIYA Studio × 紫荆文化集团
 
 AIYA Studio 是一个基于多 AGENT 制片认知封装的企业内部 AI 工作流系统。核心价值是把导演、分镜、视觉、prompt 能力封装成流程，让组织级生产成为可能。
 
 Enterprise-grade AI production workflow powered by multi-AGENT cognitive encapsulation.

 ## 关于 AIYA Studio
 
 系统完成度：80-85%
 
 核心能力：
 - 8 个 AGENT 角色组成生产闭环
 - 多角色互相质检、迭代收敛
 - 剧本讨论 → 分镜 → prompt → 资产生成 → 视频输出 全链路
 
 技术底座：GPT-Image 2 Edit + Dreamina / Seedance 2.0 + Multi AGENT Workflow
 
 ## 合作方向
 
 面向紫荆文化集团四大业务板块：
 - 影视产业（银都机构、港湾影视）
 - 出版发行（联合出版集团：三联/中华/商务）
 - 传媒资讯（紫荆杂志、中国基金报）
 - 文旅演艺（中国对外文化集团）
