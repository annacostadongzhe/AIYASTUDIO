01 · POSITIONING

AIYA Studio 不是又一个 AI 工具

它是基于 GPT-Image 2 Edit + Dreamina / Seedance 2.0 的企业内部 AI 制片工作流系统。核心不是"能生成什么"，而是能把专业制片判断交给系统流程。

一句话定义

核心价值：多 AGENT 制片认知封装——把导演、分镜、视觉、prompt 能力封装成流程，让组织级生产成为可能。

不是通用平台

不是无限画布、不是单 AGENT prompt 工具。它把专业制片判断封装成可确认的流程步骤。

认知封装

镜头语言、构图、表演、资产引用、质量门禁——这些专业能力被 AGENT 承担，不丢给操作员。

系统完成度 80-85%

核心生产链路和 AGENT 设计已经成形。已具备内部部署演示基础。

系统完成度 80-85%。内网部署方向下，核心生产链和 AGENT 设计已经成形。

02 · SYSTEM INTELLIGENCE

多 AGENT 生产判断网络

真正要评估的是 AGENT 是否能在讨论阶段持续提出、反驳、修复并收敛，而不是只会生成一段 prompt。

01Story Director

负责故事结构、冲突推进、角色动机和叙事节奏，避免剧本只停留在用户原始描述。

02Visual Director

负责画面语言、构图、色彩、空间关系和镜头可视化，把文字转成可执行的视觉判断。

03Performance Director

负责表演、停顿、动作、情绪递进，让角色不是"站着说台词"。

04Actor AGENT

带着角色身份参与讨论，补充人物反应、心理边界和表演细节。

05HIDE

决策门禁和用户意图承接层，把确认、反驳、修正转成流程状态。

06Storyboarder

把讨论结果拆成 SHOT，写入镜头目标、标注和下游生成要求。

07Lighting

补足光线、氛围、空间层次，避免画面只有主体没有电影感。

08Prompt Engineer

把资产、参考、镜头目标、质量约束封装成 SHOT 级 prompt。

关键区别

单 AGENT 是"你问它答"；AIYA 是多个制片角色互相质检，再把结果落到可生产的 SHOT。

03 · BUSINESS ALIGNMENT

对紫荆文化集团 四大业务板块 的价值

评估口径必须从"能不能生成"转成"能不能把专业制片判断交给系统流程"。

🎬 影视产业

核心切入点 · 匹配度：极高

紫荆资产：银都机构（长城电影、凤凰影业、新联影业）、港湾影视集团（筹）
AIYA 价值：剧本讨论 → 分镜生成 → 镜头设计 → 资产生成 → 视频输出 全链路闭环
关键判断：人力从创作降为审稿；低经验操作员可产出高于自身水平的结果。
行业定位：当前方案（Adobe Firefly Boards / Boords / Katalist）停留在分镜协作层面。

📚 出版发行

IP 可视化 · 匹配度：高

紫荆资产：联合出版集团（三联书店、中华书局、商务印书馆）
AIYA 价值：文字内容到视觉内容的批量转化、IP 可视化、概念图与插画生产、系列化作品的视觉一致性管理
资产优势：角色/场景/道具资产可复用，确保长线 IP 跨介质保持视觉连贯性。

📰 传媒资讯

内容生产提效 · 匹配度：中高

紫荆资产：紫荆杂志社、中国基金报社
AIYA 价值：媒体视觉内容生产提速、视频化转型、系列封面/插画标准化生产
迁移路径：SHOT 级 prompt 与参考绑定机制可迁移至图文生产场景。

🎭 文旅演艺

跨境文化 · 匹配度：中高

紫荆资产：中国对外文化集团
AIYA 价值：演出视觉概念设计、沉浸式内容制作、跨文化多语言版本制作
天然适配：中英双语原生支持，适合港澳台和东南亚跨境文化传播。

双总部战略匹配

紫荆文化集团实行深圳+香港"双总部"管理，业务覆盖港澳台和东南亚。AIYA Studio 天然适配跨境业务场景：中英双语原生支持、可私有化部署、符合国产化信创要求。

04 · BUSINESS VALUE

核心商业价值：降低组织用人成本

重点不是省几个按钮，而是来自岗位替代、培训周期缩短、产能复制和内部资产沉淀。
维度传统方案AIYA 方案
操作员要求高级复合人才：同时理解 prompt、分镜、构图、灯光、角色一致性、模型修复和视频生成低经验操作员 + 系统流程：主要负责任务确认、候选审核、补充修改
培训周期招聘稀缺复合人才，培训慢、复制难3 天上手内部工具培训，专业判断压缩为可确认步骤
产能可复制性依赖个人能力，难以批量复制多 AGENT + 资产体系 + 质量门禁，可以训练新人替代高级岗位
资产沉淀项目间资产难以复用，风格漂移角色/场景/道具资产系统化，跨项目一致性保障
失败返工依赖个人排查，效率低下自动补缺、候选重试、质量门禁闭环，不让流程断掉

差距不只是效率，而是组织可复制成本。一个系统化流程可以训练外人；一个高级复合人才很难批量复制。

— AIYA Studio System Evaluation · 09 BUSINESS VALUE

06 · MARKET BENCHMARK

行业定位：不是画布工具，是生产系统

核心对标不是"谁有画布"，而是谁能把多 AGENT 制片认知封装成企业内部生产流程。
平台类型代表产品核心定位与本系统差异
视觉协作容器Adobe Firefly Boards无限画布 + 视觉 brainstorming要会拆解、摆放、判断；只是创作容器
分镜故事板Boords / Katalist分镜生成和故事板协作偏单点辅助，不封装完整制片认知
视频生成Runway视频模型和创意生成强在模型，不等于企业内部工作流
脚本到视频LTX Studio脚本到视频的 AI Studio偏平台化操作，不是私有化生产系统
AIYA Studio本系统企业内部多 AGENT 制片工作流低经验操作员 + 审稿确认；完整封装制片认知
