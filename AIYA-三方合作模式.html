02 · COOPERATION SCOPE

合作内容

结合现有三方合作协议（寒川子项目）的结构，AIYA 项目的合作内容建议如下：

合作模块内容说明对应紫荆业务线
AI 制片工作流系统部署AIYA 系统以私有化方式部署于紫荆内部，覆盖剧本讨论到成片输出的全链路影视（银都机构、港湾影视）
IP 视觉化生产利用多 AGENT 工作流将出版 IP 批量转化为视觉内容（概念图、插画、视频）出版（联合出版集团）
媒体内容视频化为杂志和新闻内容建立标准化视频生产流程传媒（紫荆杂志、中国基金报）
文旅视觉内容制作演出视觉设计、沉浸式内容制作、多语言版本输出文旅（中国对外文化集团）
人才引进与团队整合AIYA 团队整体引入紫荆体系，保留技术核心能力组织建设

参考依据

本合作框架参考《中华商务贸易有限公司 · 中华书局（香港）有限公司 · 天下纵横（海口）文化产业投资有限公司 · 三方合作协议》的结构设计，合作内容模块与寒川子项目的"出版发行 + 读书会 + 文化出海 + IP 开发 + 海外杂志"模式对应。

04 · IMPLEMENTATION PHASES

分阶段实施路径

参照紫荆体系现有的合作节奏，建议三阶段推进：

Phase 1 · 1-2 月

技术验证与试点

选定一条业务线（建议影视）进行定向演示和私有部署 POC。完成剧本 benchmark、样片生产和 SOP 编写。

Phase 2 · 2-4 月

正式合作启动

签署三方合作协议，完成系统正式部署。甲方提供业务场景，丙方协调资源，AIYA 交付生产标准。

Phase 3 · 4-6 月

全业务推广

从试点业务线扩展至出版、传媒、文旅板块。建立资产体系和培训机制，探索人才引进路径。

05 · REVENUE MODEL

收益分配机制

参照现有三方协议的"经费及收益分配"条款，AIYA 项目的收益模式建议如下：

基础服务费

系统私有化部署的年度许可费 + 首次实施费。甲方承担，丙方协调预算审批。

项目收益分成

针对具体制片/生产项目，按项目收益或制作预算约定比例分成。三方根据贡献度协商分配比例。

IP 开发收益

利用 AIYA 系统产出的 IP 衍生内容产生的收益（如视频版权、数字资产、视觉内容授权），三方另行约定分配。

服务增值费

超出基础服务的定制开发、模型调优、额外培训等产生的费用，按实际投入计价。

06 · IP & ASSET OWNERSHIP

知识产权与资产归属

参照现有协议的知识产权条款，AIYA 项目需要明确的归属界定：

资产类型归属
AIYA 系统本身（代码、架构、AGENT 设计）归 AIYA Studio（乙方）团队所有
合作过程中产出的视觉资产（图像、视频、数字内容）归甲方所有，乙方保留技术署名权
合作过程中产生的系统优化（定制功能、工作流改进）三方共同所有，乙方可复用于其他项目
培训材料和 SOP 文档归甲方所有，乙方保留副本参考

参考

寒川子协议约定："合作过程中产生的新的知识产权（如翻译作品、IP 衍生品等），由三方共同所有，具体权益比例由三方另行协商确定。" AIYA 项目的 IP 归属可参照此结构。

07 · NEXT STEPS
